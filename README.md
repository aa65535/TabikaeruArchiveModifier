使用说明
---

三叶草数量最大可设置为 `999999999`

抽奖券数量最大可设置为 `999`

可以准确调节游戏音量（游戏设置里为 20 一格, 进入游戏设置后可能会被修改）

可以将明信片无损导出到 `/sdcard/Pictures/Tabikaeru`

可以将邮箱里的广告改成回礼(99三叶草), 解决广告打不开的问题.
